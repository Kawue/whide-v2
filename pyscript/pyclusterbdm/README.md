pycluster
=========

Python library for (unsupervised) learning.

####Algorithms provided:

1. H2SOM
2. SOM
3. K-Means
4. Neural Gas
5. K-Nearest Neighbor

####Cluster metrics provided:

1. Calinski Harabasz Index
2. indexI
3. seperation
4. silhouetteWidth
5. Davies Bouldin Index
6. Dunn Index

#### Installation

`python setup.py install`
